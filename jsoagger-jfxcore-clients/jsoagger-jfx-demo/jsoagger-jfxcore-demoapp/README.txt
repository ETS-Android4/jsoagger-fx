This is the demo application of EmaginFX.
--------------------------------------------------------

This demo application shows a partial view of its capabilities and builtin components including:

	- Table view
	- List view
	- Preferences mangement
	- Form in view mode
	- Form in edit mode
	- Wizard
	- Details view
	- Views navigation
	- Theme management
	- Search management
	- Responsive
	
	- Layouts management
	- Mobile layout
	- Xpad layout


Running
--------------------------------------------------------
1. JavaFX runtime must be installed
2. Launch script run.bat or run.sh
